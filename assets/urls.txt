Here’s a 15-question JavaScript final exam. Some questions include code examples with 4 answer options and one correct answer:

---

### **1. What is the correct syntax to declare a JavaScript variable?**  
a. `var myVariable;`  
b. `variable myVariable;`  
c. `let myVariable =;`  
d. `const myVariable == 5;`  

**Correct answer**: a. `var myVariable;`

---

### **2. What will the following code output?**  
```javascript
console.log(typeof 42);
```
a. `string`  
b. `number`  
c. `undefined`  
d. `object`  

**Correct answer**: b. `number`

---

### **3. How do you write a single-line comment in JavaScript?**  
a. `<!-- This is a comment -->`  
b. `# This is a comment`  
c. `// This is a comment`  
d. `* This is a comment`  

**Correct answer**: c. `// This is a comment`

---

### **4. What will the following code output?**  
```javascript
let x = 5;
let y = '5';
console.log(x == y);
```
a. `true`  
b. `false`  
c. `undefined`  
d. `Error`  

**Correct answer**: a. `true`

---

### **5. Which of these array methods adds an item to the end of an array?**  
a. `push()`  
b. `pop()`  
c. `shift()`  
d. `unshift()`  

**Correct answer**: a. `push()`

---

### **6. What will the following code output?**  
```javascript
function test() {
    console.log(this);
}
test();
```
a. `undefined`  
b. `null`  
c. `Window` or `global object`  
d. `test`  

**Correct answer**: c. `Window` or `global object`

---

### **7. What is the result of this code?**  
```javascript
let a = 10;
console.log(a++);
```
a. `10`  
b. `11`  
c. `undefined`  
d. `Error`  

**Correct answer**: a. `10`

---

### **8. What will the following code output?**  
```javascript
console.log(0.1 + 0.2 === 0.3);
```
a. `true`  
b. `false`  
c. `undefined`  
d. `Error`  

**Correct answer**: b. `false`

---

### **9. How can you define a function in JavaScript?**  
a. `function myFunction() {}`  
b. `let myFunction = function {};`  
c. `func myFunction() {}`  
d. `define myFunction() {}`  

**Correct answer**: a. `function myFunction() {}`

---

### **10. What will the following code return?**  
```javascript
const fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]);
```
a. `"apple"`  
b. `"undefined"`  
c. `Error`  
d. `"cherry"`  

**Correct answer**: b. `"undefined"`

---

### **11. Which of the following correctly implements a for loop?**  
a.  
```javascript
for i = 0; i < 5; i++ {
    console.log(i);
}
```
b.  
```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```
c.  
```javascript
for (i < 5; i++) {
    console.log(i);
}
```
d.  
```javascript
for (let i = 0, i < 5, i++) {
    console.log(i);
}
```

**Correct answer**: b.  

---

### **12. What does the `Array.prototype.map()` method do?**  
a. It creates a new array with the results of calling a provided function on every element.  
b. It changes the original array by applying a function to each element.  
c. It filters elements from the array.  
d. It sorts the array in ascending order.  

**Correct answer**: a.

---

### **13. What will the following code output?**  
```javascript
const obj = { name: "John" };
const obj2 = obj;
obj2.name = "Jane";
console.log(obj.name);
```
a. `"John"`  
b. `"Jane"`  
c. `undefined`  
d. `Error`  

**Correct answer**: b. `"Jane"`

---

### **14. How do you check if a variable is `null` in JavaScript?**  
a. `typeof variable === "null"`  
b. `variable === null`  
c. `variable == undefined`  
d. `typeof variable == undefined`  

**Correct answer**: b. `variable === null`

---

### **15. What will the following code output?**  
```javascript
setTimeout(() => {
    console.log("Hello");
}, 0);
console.log("World");
```
a. `"Hello"`, then `"World"`  
b. `"World"`, then `"Hello"`  
c. `Error`  
d. `undefined`  

**Correct answer**: b. `"World"`, then `"Hello"`
